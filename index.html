Quizzical

Quizzical ist eine Quiz-App, die mit React erstellt wurde und Fragen von der Open Trivia Database API abruft. Die App ist dafür konzipiert, das Wissen des Benutzers in der Kategorie "Allgemeinwissen" zu testen. Der Benutzer kann für jede Frage eine Antwort auswählen und nach Beantwortung aller Fragen seine Punktzahl überprüfen.

Tenzies

Dieses Projekt ist eine React-basierte Webanwendung, die ein einfaches Würfelspiel namens Tenzies implementiert. Ziel des Spiels ist es, zehn Würfel zu werfen, bis alle die gleiche Augenzahl anzeigen. Der Benutzer kann sich dafür entscheiden, einen Würfel "festzuhalten", was bedeutet, dass er beim nächsten Zug nicht erneut gewürfelt wird. Wenn der Benutzer das Spiel gewinnt, indem alle zehn Würfel die gleiche Augenzahl zeigen, wird eine Konfetti-Animation, sowie die Zeit und Anzahl der Würfe, die benötigt wurden, um zu gewinnen angezeigt.

Passwort Generator

Dieses Projekt ist eine webbasierte Anwendung, die fünf zufällige Passwörter auf der Grundlage von vom Benutzer spezifizierten Optionen generiert. Die Passwörter werden unter Verwendung einer Kombination aus Klein- und Großbuchstaben, Zahlen und Symbolen generiert. Die App wurde mit HTML, CSS und JavaScript erstellt. Sie verfügt über eine einfache Benutzeroberfläche, die es dem Benutzer ermöglicht, die Passwortlänge sowie Optionen für die Einbeziehung von Großbuchstaben, Zahlen und Symbolen auszuwählen.

Wetter App

Dieses Projekt ist eine Webanwendung, die Wetterdaten auf Basis des vom Benutzer eingegebenen Stadtnamens abruft und anzeigt. Die App verwendet die OpenWeatherMap-API, um die Daten für die Stadt abzurufen und zeigt die Temperatur, Luftfeuchtigkeit, Windgeschwindigkeit und das Wettericon an. Das Hintergrundbild der App ändert sich je nach gesuchter Stadt.
