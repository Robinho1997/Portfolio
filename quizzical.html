Quizzical

Dieses Projekt ist eine Quiz-App, die Fragen von der Open Trivia Database API abruft. Die App ist dafür konzipiert, das Wissen des Benutzers in der Kategorie "Allgemeinwissen" zu testen. Der Benutzer kann für jede Frage eine Antwort auswählen und nach Beantwortung aller Fragen seine Punktzahl überprüfen.
Die App wurde mit functional React-Components erstellt.

Funktionalität:
Die Hauptfunktionalität des Projekts ist in der App-Komponente implementiert, die als Standardkomponente exportiert wird. Sie enthält die State Variablen started, questions, count, checked und correct.

Die State Variable "started" speichert den aktuellen Zustand des Quiz, d. h. ob es gestartet wurde oder nicht. Die "questions" State Variable ist ein Array, das fünf Frageobjekte enthält, die von der Open Trivia Database API abgerufen wurden. Jedes Frageobjekt hat eine ID, Antworten (eine gemischte Liste aller möglichen Antworten), Frage, Correct (die richtige Antwort), Selected (die vom Benutzer ausgewählte Antwort) und Checked (ein Boolescher Wert, der angibt, ob der Benutzer seine Antwort ausgewählt hat).

Die "count" State Variable speichert die Anzahl der Rendervorgänge der Komponente und wird verwendet, um den useEffect-Hook auszulösen, der neue Fragen von der API abruft, wenn der Benutzer auf die Schaltfläche "Erneut spielen" klickt.

Die "checked" State Variable speichert anhand einer boolean den Zustand, ob der Benutzer seine Antworten ausgewählt hat. Die "correct" Variable speichert die Anzahl der vom Benutzer ausgewählten korrekten Antworten.

Die Komponente enthält mehrere Funktionen, die die Quiz-Logik implementieren. Die "shuffleArray" Funktion mischt das Antwortarray jedes Frageobjekts. Die "handleCheck" Funktion überprüft, ob der Benutzer alle Fragen beantwortet hat, und setzt die "checked"- und "correct" State Variablen. Die "handleClickAnswer" Funktion aktualisiert die ausgewählte Eigenschaft des Frageobjekts, wenn der Benutzer eine Antwort auswählt. Die "handlePlayAgain" Funktion setzt die "checked" State Variable zurück und erhöht die "count" State Variable, um neue Fragen von der API abzurufen.

App.jsx enthält auch einen useEffect-Hook, der Fragen von der API abruft, wenn sich die count Variable ändert.

Die Komponente rendert entweder die StartPage- oder Question-Komponenten, abhängig vom Wert der "started" State Variable. Die Question-Komponente zeigt jede Frage mit ihren Antwortoptionen an und ermöglicht dem Benutzer, eine Antwort auszuwählen. Nachdem der Benutzer alle Fragen beantwortet hat, werden die Punktzahl und die Schaltfläche "Erneut spielen" oder "Antwort überprüfen" angezeigt.


Fazit:
Dieses Projekt demonstriert den Einsatz von React-Zustandsvariablen, useEffect-Hooks und Ereignisbehandlung zum Erstellen einer Quiz-App, die Fragen von einer externen API abruft. Es bietet ein Beispiel dafür, wie man Arrays mischt und Eigenschaften von Objekten in React-Komponenten aktualisiert.
