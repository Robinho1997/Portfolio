Tenzies Würfel Spiel

Dieses Projekt ist eine React-basierte Webanwendung, die ein einfaches Würfelspiel namens Tenzies implementiert. Ziel des Spiels ist es zehn Würfel zu werfen bis alle die gleiche Augenzahl anzeigen. Der Benutzer kann sich dafür entscheiden einen Würfel "festzuhalten", was bedeutet dass er beim nächsten Zug nicht erneut gewürfelt wird. Wenn der Benutzer das Spiel gewinnt indem alle zehn Würfel die gleiche Augenzahl zeigen erfolgt eine Konfetti-Animation. Darüber hinaus wird die Zeit und die Anzahl der Würfe angezeigt, die der Benutzer benötigt hat.

Projektübersicht

Funktionalität:
Die Hauptfunktionalität des Projekts wird in der App-Komponente implementiert, die als Standardkomponente exportiert wird. Sie enthält die State Variablen count, dice, tenzies, startGame und endGame. Die "count" State Variable speichert die Anzahl der Würfe des Benutzers. Die "dice" State Variable ist ein Array welches zehn Würfelobjekte enthält, von denen jedes eine "value", "isHeld" und "Id" Eigenschaft hat. Die "tenzies" State Variable ist ein boolescher Wert, der anzeigt, ob der Benutzer das Spiel gewonnen hat. Die "startGame" und "endGame" State Variablen speichern die Zeit, zu der das Spiel begann und endete.

"App.jsx" enthält mehrere Funktionen, die die Spiellogik implementieren. Die Funktion "allNewDice" generiert ein neues Array mit zehn Würfelobjekten mit zufälligen Werten und die Funktion "generateNewDie" generiert ein einzelnes Würfelobjekt mit einem zufälligen Wert zwischen 1 und 6. Die "rollDice" Funktion würfelt die Würfel, indem sie neue Würfel für alle Würfel generiert, die nicht gehalten werden, und inkrementiert die "count" State Variable. Wenn der Benutzer das Spiel gewonnen hat, setzt die Funktion das Spiel zurück indem sie neue Würfel generiert und die "count", "startGame" und "tenzies" State Variablen zurücksetzt. Die Funktion "holdDice" toggelt die "isHeld" Eigenschaft eines Würfelobjekts mithilfe der angegebenen Id.

Die Komponente enthält auch einen useEffect-Hook, der jedes Mal ausgeführt wird wenn sich die "dice" State Variable ändert. Dieser Hook prüft, ob alle Würfel gehalten werden und die gleiche Augenzahl haben und setzt gegebenenfalls die "tenzies" und "endGame" State Variablen auf einen neuen Wert.

Zudem rendert die Komponente eine Überschrift, Anweisungen, Würfel-Elemente und einen Button. Wenn der Benutzer das Spiel gewonnen hat, werden eine Konfetti-Animation und eine Nachricht angezeigt, welche die Zeit und die Anzahl der Würfe darsellt die benötigt wurden um zu gewinnen.


Fazit:
Dieses Projekt ist eine einfache aber unterhaltsame Implementierung des Tenzies-Würfelspiels mit React. Es demonstriert die Verwendung von Zustandsvariablen, useEffect-Hooks und event-handling in React und bietet ein Beispiel dafür wie man ein einfaches Spiel mit React-Komponenten implementieren kann.
