Passwort Generator

Dieses Projekt ist eine webbasierte Anwendung welche fünf zufällige Passwörter auf der Grundlage von vom Benutzer spezifizierten Optionen generiert. Die Passwörter werden unter Verwendung einer Kombination aus Klein- und Großbuchstaben, Zahlen und Symbolen generiert. Die App wurde mit HTML, CSS und JavaScript erstellt. Sie verfügt über eine einfache Benutzeroberfläche, die es dem Benutzer ermöglicht, die Passwortlänge sowie Optionen für die Einbeziehung von Großbuchstaben, Zahlen und Symbolen auszuwählen.

Funktionalität:
Der JavaScript-Code für die App ist in mehrere Funktionen unterteilt. Die Funktion "generateFivePasswords" ist die Hauptfunktion, die aufgerufen wird, wenn der Benutzer auf die Schaltfläche "Generieren" klickt. Zunächst wird der Passwort-Container gelöscht, indem sein "innerHTML" auf einen leeren String gesetzt wird. Dann wird die Passwortlänge aus dem Element "lengthInput" abgerufen und unter Verwendung der Funktion "generateCharset" ein Charset auf Basis der vom Benutzer ausgewählten Optionen generiert. Anschließend werden fünf Passwörter unter Verwendung einer for Schleife generiert, die zufällige Zeichen aus dem Charset zusammensetzt. Jedes Passwort wird in dem Array passwordsArray gespeichert. Schließlich werden die Passwörter in dem Passwort-Container angezeigt, indem für jedes Passwort ein neues p-Element erstellt und dem Container angehängt wird.

Die Funktion "generateCharset" generiert ein Charset auf Basis der vom Benutzer ausgewählten Optionen für Großbuchstaben, Zahlen und Symbole. Zunächst wird das Charset nur mit Kleinbuchstaben und Zahlen initialisiert. Anschließend werden mit "if-else"-Statements alle Kontrollkästchen überprüft und die entsprechende Zeichenmenge dem Charset hinzugefügt.

Fazit:
Insgesamt zeigt diese App, wie man JavaScript verwendet, um zufällige Passwörter auf Basis von vom Benutzer spezifizierten Optionen zu generieren.
