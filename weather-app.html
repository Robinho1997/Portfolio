Wetter App

Dieses Projekt ist eine Webanwendung, welche Wetterdaten auf Basis des vom Benutzer eingegebenen Stadtnamens abruft und anzeigt. Die App verwendet die OpenWeatherMap-API, um die Daten für die Stadt abzurufen und zeigt die Temperatur, Luftfeuchtigkeit, Windgeschwindigkeit und ein Wettericon an. Das Hintergrundbild der App ändert sich je nach gesuchter Stadt.

Projektübersicht

Funktionalität:
Der JavaScript-Code enthält ein Objekt namens "weather", das drei Methoden enthält: fetchWeather, displayWeather und searchWeather.

fetchWeather: Diese Methode nimmt den vom Benutzer eingegebenen Stadtnamen entgegen und ruft die Wetterdaten für diese Stadt über die OpenWeatherMap-API ab. Sie verwendet die apiKey-Eigenschaft des "weather"-Objekts zur Authentifizierung des API-Aufrufs. Sobald die Daten abgerufen sind, ruft sie die Methode "displayWeather" auf und übergibt die Daten als Argument.

displayWeather: Diese Methode nimmt die vom API abgerufenen Wetterdaten entgegen und zeigt sie auf der Webseite an. Sie extrahiert die relevanten Daten aus der Antwort mit Hilfe der Destrukturierung und aktualisiert die entsprechenden Elemente des HTML-DOMs mit den Daten.

searchWeather: Diese Methode wird aufgerufen, wenn der Benutzer auf die Such-Schaltfläche klickt oder nach der Eingabe des Stadtnamens in der Suchleiste die Eingabetaste drückt. Sie ruft die Methode "fetchWeather" mit dem Wert des Suchleisten-Inputs als Argument auf.

Beim Laden der Seite wird die Methode "fetchWeather" mit einem Standardwert von Berlin ausgeführt.

Fazit:
Zusammenfassend demonstriert dieses Projekt die Fähigkeit, eine externe API in eine Webanwendung zu integrieren und die Daten so zu manipulieren, dass sie benutzerfreundlich angezeigt werden. Die Verwendung von objektorientierter Programmierung mit dem "weather"-Objekt und seinen Methoden verbessert die Lesbarkeit und Organisation des Codes. Die Event-Listener für die Suchleiste und die Suchschaltfläche sorgen für eine nahtlose Benutzererfahrung. Insgesamt präsentiert dieses Projekt die Fähigkeiten und Kenntnisse, die erforderlich sind, um eine funktionale und interaktive Webanwendung mit HTML, CSS und JavaScript zu erstellen.
